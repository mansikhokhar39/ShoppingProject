# Shopping Cart Application

This is a full-stack shopping cart application built using React, Node.js, Express, and MongoDB.

## Features
- User authentication (Signup & Login)
- Product listing with images
- Add to cart and quantity management
- Checkout with address input
- Order placement with success confirmation
- Order history view

## Tech Stack
- Frontend: React, CSS
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## How to Run
1. Clone the repository
2. Run backend using `npm start`
3. Run frontend using `npm start`
4. Open browser at `http://localhost:3000`

This project demonstrates a complete shopping cart flow from login to order placement.
